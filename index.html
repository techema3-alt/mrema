why termux-first

Built for the phone, not ported to it.

Recon suites are usually written for a laptop and then apologised for on Android. This one was written inside Termux from the first line.

▮▮▯

No root, no VPS

Every module runs on-device. Nothing phones home, nothing needs a server you have to rent.

▮▮▮

Signal-cyan UI

A high-contrast Rich-powered interface, readable in direct sunlight on a five-inch screen.

▮▯▯

ARM-safe installs

Dependencies are pinned to versions that build cleanly on Termux's ARM Python, no Rust toolchain required.

get running

Three commands to first scan.

termux
requirements
# clone the suite
$ pkg install python git -y
$ git clone https://github.com/techema3-alt/mr-ema-recon-suite
$ cd mr-ema-recon-suite

# install deps (ARM-safe)
$ pkg install python-cryptography -y
$ pip install -r requirements.txt

# run it
$ python mr_ema_recon.py
